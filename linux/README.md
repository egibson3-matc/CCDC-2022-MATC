# Linux README.md

### iptables.sh

Note: This script must be run with root privileges.
________________
Steps:
1. Make the file executable. 
```
sudo chmod u+x iptables.sh
```
2. Run the script.
```
sudo ./iptables.sh
```
3. Select multiple options separated by a space ' ' 
- Example command:
```
1 4 6
```
